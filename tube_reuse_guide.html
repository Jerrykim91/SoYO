상세 재사용 안내

1. 내용물 비우기

튜브 안에 남아 있는 제형을 최대한 끝까지 비워 주세요. 잔여물이 적을수록 더 깔끔하고 위생적으로 다시 사용할 수 있어요.

2. 충분히 세척하기

미온수로 여러 번 헹궈 내부에 남은 내용물을 제거해 주세요. 입구 부분과 캡 안쪽도 함께 닦아 주면 더 좋아요.

3. 완전히 건조하기

세척 후에는 물기를 충분히 털어내고, 내부까지 완전히 건조해 주세요. 습기가 남아 있으면 새로 담는 내용물의 상태에 영향을 줄 수 있어요.
TIP: 건조하고 알콜로 한번 소독해주면 더욱 위생적으로 사용할 수 있습니다.

4. 이렇게 활용해 보세요

  • A
    여행용 소분 용기 : 클렌저, 바디워시 등을 소량 담아 휴대
  • B
    리필 공용기 : 자주 쓰는 제품을 덜어 담아 사용
  • C
    셀프 케어 용기 : 간단한 생활용 제품이나 개인용 아이템 보관

재사용 시 주의사항

용기 변형이나 마모가 심한 경우에는 재사용을 멈추고 새로운 용기를 사용해 주세요. 위생이 중요한 내용물을 담을 때에는 세척과 건조 상태를 꼭 확인해 주세요.
